CITEgeist Documentation
=======================

Welcome to CITEgeist's documentation!

CITEgeist is a computational framework for spatial transcriptomics analysis that combines gene expression and antibody capture data to deconvolve cell type proportions and gene expression profiles at single-cell resolution.
